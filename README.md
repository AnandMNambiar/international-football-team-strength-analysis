# International Football Team Strength Analysis

## Overview
This project analyzes over 49,000 international football matches from 1872 to 2026 using Python, Pandas, and Matplotlib.

The goal was to identify the strongest international football teams based on:
- Win Percentage
- Goal Difference

## Dataset
International Football Results Dataset (1872–2026)

## Technologies Used
- Python
- Pandas
- Matplotlib

## Project Workflow
1. Data loading and exploration
2. Data cleaning
3. Handling missing values
4. Team performance analysis
5. Win percentage calculation
6. Goal difference calculation
7. Team strength score creation
8. Data visualization

## Top Ranked Teams
1. Spain
2. Brazil
3. Argentina
4. Belgium
5. Portugal
6. England
7. Germany
